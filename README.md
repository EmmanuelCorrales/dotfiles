# vim-config
<p>My vim configuration.</p>

<h2>Installation</h2>
<p>Run <b>./setup</b>.</p>


<h2>Typescript</h2>
<p>Run <b>./typescript-setup</b>.</p>
<h3>Warning</h3>
<p><b>typescript-setup</b> installs the plugin <a href='https://github.com/Quramy/tsuquyomi'>tsuquyomi</a>.
  I noticed that it made vim open ts files slower.
</p>
